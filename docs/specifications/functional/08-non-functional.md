# 8. Non-functional expectations

---

## 8.1 Performance

Heavy libraries (Prettier, LESS, Lodash chunks, Ace) load **on demand**; expect Suspense/loading UI while chunks load. See [08-text-utils-and-lazy-loading.md](../technical/08-text-utils-and-lazy-loading.md).

---

## 8.2 Accessibility

Native `<select>` and icon buttons with `title`. Full WCAG compliance is **not** specified as a requirement in this repo.

---

## 8.3 Internationalization

UI strings are **English** only in source.
